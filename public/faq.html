 Which data did you use?
Which data did you use? The data on habitats, their sufficiency, CO2-equivalent-emissions and the original maps where taken from eurostat, provided by the European Commision. If you want to get the original data you can download the following databases, just search here (http://epp.eurostat.ec.europa.eu/portal/page/portal/statistics/search_database ) for the databases env_bio1 and env_air_gge.
